# AI Credit Risk Analyst

Агент-аналитик кредитного риска, помогающий риск-аналитику быстро анализировать заемщика и объяснять риск.

---

## Задача, аудитория и боль

**Задача:** Ускорить и стандартизировать первичный анализ кредитных заявок.

**Для кого:** Риск-аналитики банка, обрабатывающие поток заявок на кредитование.

**Боль сейчас:**
- Ручной сбор данных из разных источников (БКИ, скоринг, политика) занимает значительное время
- Субъективность в интерпретации рисков и формулировке выводов
- Долгий цикл от поступления заявки до рекомендации
- Высокая нагрузка на аналитиков при росте объёма заявок

---

## Что именно сделает PoC на демо

- Агент получает запрос аналитика (например: «Оцени клиента 12345 для кредита 500k»)
- Запрашивает данные из API кредитной истории и внутренней базы скорингов (PD-модель)
- Проверяет клиента по кредитной политике банка через RAG
- Формирует risk-assessment report с объяснением причин и рекомендацией (approve / reject / manual review)
- Демонстрирует обработку edge cases: нет кредитной истории, конфликт данных, таймаут PD-модели, превышение лимита

---

## Что НЕ делает PoC (out-of-scope)

- Реальные интеграции с production API БКИ и скоринговых систем — используются mock-данные
- Финальное решение по выдаче кредита — агент только рекомендует, решение принимает человек
- Массовая пакетная обработка заявок
- Сертификация для production в регулируемой среде
- Полноценный диалоговый интерфейс с историей сессий
- Интеграция с CRM и учётными системами банка
