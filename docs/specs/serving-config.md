# Spec: Serving / Config

## Запуск (PoC)

| Режим | Описание |
|-------|----------|
| Локальный | Один процесс API + worker оркестратора (или монолит) |
| Контейнер | Dockerfile опц.; переменные окружения из `.env` (не коммитить) |

## Конфигурация (примеры ключей)

| Ключ | Назначение |
|------|------------|
| `LLM_MODEL` | Имя модели (фиксировать версию для воспроизводимости) |
| `LLM_MAX_TOKENS` | Верхняя граница ответа |
| `EMBEDDING_MODEL` | Модель для индекса политики |
| `CREDIT_HISTORY_TIMEOUT_MS` | Таймаут БКИ |
| `PD_TIMEOUT_MS` | Таймаут PD |
| `RETRIEVER_TOP_K` | k для vector search |
| `CONTEXT_MAX_TOKENS` | Общий budget для промпта |

## Секреты

- `OPENAI_API_KEY` / аналог — только через env или secret manager; **не** в репозитории.
- Ключи к внутренним API — те же правила.

## Версии моделей

- Зафиксировать в `README` или `.env.example` **имена и версии** моделей, используемых в PoC.
- При смене модели — переиндексировать политику при смене embedding-модели.

## Feature flags (PoC)

| Флаг | Назначение |
|------|------------|
| `RERANKER_ENABLED` | Вкл/выкл rerank после retrieval |
| `MOCK_EXTERNAL_APIS` | Моки БКИ/PD для разработки без внешних вызовов |
